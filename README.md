# Python Log Format Generator

This tool will help you customize the format of your python log messages.

## Setup

Installation

```sh
npm install
```

Development server

```sh
npm run dev
```

Type checking, compiling and minification for production

```sh
npm run build
```

Linting

```sh
npm run lint
```
