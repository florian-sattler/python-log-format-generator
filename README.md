# Python Log Format Generator

Python Log Format Generator is an open-source web application written in Vue.js that provides a convenient way to generate format strings for [Python logging](https://docs.python.org/3/library/logging.html).
With this tool, users can effortlessly create custom log formats by selecting from presets or starting from scratch.

## Features

- **All three format styles**: Build formats in any of Python's `logging.Formatter` styles — `%`-style (`%(levelname)s`), `{`-style / `str.format` (`{levelname}`), and `$`-style / `string.Template` (`$levelname`).

- **Full field formatting**: Configure width, alignment, precision, sign, fill, thousands grouping and the conversion/presentation type per field (e.g. `%(levelname)-8s`, `{lineno:>4}`, `{created:,.2f}`), plus a custom `datefmt` for `asctime`.

- **Accurate live preview**: As you build the format, the output is previewed against sample log records using the real Python formatting rules. The engine is verified against actual Python 3.12 output in CI, so the preview matches what `logging` produces.

- **Instant feedback & clipboard**: See the resulting format string immediately and copy a ready-to-use `logging.Formatter(...)` snippet with a single click.

- **Editing capabilities**: Modify, reorder and edit existing format elements, with presets to start from.

- **Modern interface**: A clean look and feel with both dark and light themes.

## Development

To get started using the Python Log Format Generator, follow these steps:

1. Clone the repository: `git clone https://github.com/florian-sattler/python-log-format-generator.git`
2. Install the necessary dependencies: `npm install`
3. Launch the application: `npm run dev`

Useful scripts: `npm run test` runs the test suite (the formatting engine is checked against ground-truth output generated by real Python 3.12 — see `tests/oracle/`), `npm run lint` and `npm run type-check` validate the code, and `npm run build` produces the production bundle.
